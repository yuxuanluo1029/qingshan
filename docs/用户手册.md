# 用户手册

## 1. 产品简介

城迹是一个城市历史文化研学智能体。你可以通过它完成：
- 研学路线生成
- 点位地图查看
- 节点知识问答
- 学习报告输出

## 2. 使用流程

### 步骤 1：进入首页

点击“开始研学”。

### 步骤 2：设置研学参数

1. 选择城市
2. 选择时长（半日 / 一日 / 两日）
3. 选择学习偏好
4. 选择文化主题标签（最多 2 个）

点击“生成研学方案”。

### 步骤 3：与 AI 导师对话

- 查看系统给出的路线建议
- 可输入问题，例如：
  - “这个路线适合怎么讲解？”
  - “时间不够时该优先看哪几个点？”

### 步骤 4：查看研学路线

- 查看地图点位
- 查看每个节点的推荐理由和学习要点
- 点击任意节点进入详情

### 步骤 5：完成节点任务

在节点详情中完成：
- 阅读历史文化简介
- 查看知识亮点
- 执行现场观察任务
- 使用右下角 AI 问答补充知识

### 步骤 6：查看学习画像与报告

完成路线后进入：
- 学习画像（导师协同图谱）
- 研学报告（关键词、亮点、画像、下一步建议）

## 3. 常见问题

### Q1：地图无法显示怎么办？

请检查：
1. 是否配置 `VITE_TENCENT_MAP_KEY`
2. 当前访问域名是否在腾讯地图 Key 白名单

### Q2：AI 没有返回真实模型结果？

请检查后端 `.env` 是否配置：
- `TENCENT_SECRET_ID`
- `TENCENT_SECRET_KEY`

未配置时系统会自动使用 mock 回退。

### Q3：前后端分开部署时前端如何请求后端？

设置 `VITE_API_BASE_URL` 为后端公网地址。

## 4. 建议使用方式（比赛演示）

1. 先展示设置页参数选择
2. 展示 AI 导师推荐路线
3. 打开路线页地图与点位卡片
4. 进入单节点，演示任务与问答
5. 最后展示学习画像与研学报告
